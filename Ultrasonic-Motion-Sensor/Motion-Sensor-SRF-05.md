#HY-SRF05 Ultrasonic Position Sensor
###Specifications
5-Pin, One each for VCC, Trigger, Echo, Out and Ground.


1. Voltage : DC5V 
2. Static current : less than 2mA 
3. Detection distance: 2cm-450cm 
4. High precision: up to 0.3c
